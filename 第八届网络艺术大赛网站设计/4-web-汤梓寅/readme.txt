//作品基本说明

校办公室官网

网站具备三级页面，以白色和蓝色为主色调，具有良好的响应式效果，所用技术包括HTML5，CSS3, less，jQuery，Bootstrap,backbone，React，Router，PHP。

childhtml文件夹内存放二级页面
	admi.html 行政公文页面
	annc.html 通知公告页面
	download.html 下载专区页面
	rule.html 规章制度页面
	serve.html 服务指南页面
	statis.html 统计工作页面
	survey.html 部门简介页面
	telep.html 办公电话页面

grandchild文件夹中的Template.html用以生成所有三级页面
